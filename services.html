Every engagement is fixed-scope with a written proposal. Below is exactly what each service covers — and what it doesn't.

01 — mobile

Mobile App Development

From wireframe to a published Play Store listing. We handle the release process, not just the code.

  • Android apps in Kotlin, or cross-platform in Flutter
  • Backend APIs, authentication, and payments integration
  • Play Store listing, release pipeline, and crash monitoring
  • Post-launch support window included in every proposal
deliverable: source code + store listing + CI pipeline + handover doc

What we don't do.

No iOS-only native builds, no game development, no blockchain, no "AI strategy consulting" without shipping code. If your project needs one of these, we'll tell you on the first call and point you elsewhere if we can.

Saying no early is part of the service.
